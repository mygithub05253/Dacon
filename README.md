# Dacon 대회 참여 기록

데이콘(Dacon) 플랫폼의 다양한 대회에 참여하며 만든 기획서, 설계 문서, 구현 코드를 정리하는 레포지토리입니다.

각 대회별로 독립된 폴더를 생성하여 제출물과 관련 자료를 관리합니다.

---

| 2026 스마트 공장 운영 시스템 MVP 개발 해커톤 | 2026.05 | AI 품질 예측·원인 분석 플랫폼 (QualityLens) | [`smart-factory-hackathon/`](./smart-factory-hackathon/) |

---

## 폴더 구조

```
Dacon/
├── README.md                          ← 이 파일 (레포 전체 소개)
├── investment-dashboard-hackathon/    ← 대회별 폴더
│   ├── README.md                      ← 대회 개요 및 제출물 설명
│   ├── docs/                          ← 기획서 (Phase별 설계 문서)
│   ├── skills/                        ← Skills.md 규칙 문서
│   └── ...
└── (향후 대회 폴더 추가)
```

---

## 사용 방법

각 대회 폴더의 `README.md`를 읽으면 해당 대회의 개요, 평가 기준, 제출물 구성, 핵심 설계 방향을 파악할 수 있습니다.
