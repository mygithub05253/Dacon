# Phase 1. 서비스 컨셉 정의서

## 서비스명

**AlphaFolio** (알파폴리오)

> Alpha(초과수익) + Portfolio(포트폴리오)
> "당신의 포트폴리오에서 알파를 찾아주는 AI 투자 대시보드"

---

## 한 줄 소개

**투자 데이터를 올리면, AI가 분석하고 대시보드를 자동으로 만들어주는 서비스**

---

## 서비스 개요

AlphaFolio는 개인 투자자가 보유한 주식/ETF 포트폴리오 데이터를 업로드하면, Skills.md에 정의된 분석 규칙에 따라 자동으로 시각화하고 AI 인사이트를 생성하는 금융 투자 대시보드입니다.

핵심 철학은 **"규칙 기반 범용 투자 분석"**입니다. 분석 기준, 시각화 방식, 인사이트 생성 규칙을 Skills.md 문서에 분리하여 정의함으로써, 코드 수정 없이 분석 규칙만 교체하면 다양한 투자 데이터 구조에 대응할 수 있습니다.

---

## 타겟 사용자

### 1차 타겟: 개인 투자자
- 국내/미국 주식 및 ETF에 직접 투자하는 개인
- 포트폴리오 현황 파악, 수익률 분석, 리밸런싱 판단이 필요한 사용자
- 증권사 앱에서 CSV를 내려받지만 제대로 분석할 도구가 없는 사용자

### 2차 타겟: 투자 동호회 / 스터디 그룹
- 여러 명이 각자 포트폴리오를 공유하고 비교 분석하는 그룹
- 정기 모임에서 투자 리포트를 자동 생성하여 공유하려는 니즈

---

## 해결하는 문제 (Pain Point)

| 문제 | 설명 |
|------|------|
| 데이터 형식 제각각 | 증권사(키움, 미래에셋, 삼성 등)마다 CSV 형식이 다르고, 해외 브로커(토스증권, 한투 등) 역시 포맷이 상이함 |
| 분석 기준 부재 | 개인 투자자는 수익률, 섹터 분배, 리스크 지표 등 무엇을 어떻게 봐야 하는지 기준이 없음 |
| 시각화 허들 | 엑셀이나 코딩 없이는 포트폴리오를 직관적으로 시각화하기 어려움 |
| 인사이트 부족 | 데이터는 있지만 "그래서 뭘 해야 하는지" 해석이 어려움 |

---

## 제공하는 가치 (Solution)

| 가치 | 설명 |
|------|------|
| 범용 데이터 파서 | 다양한 형식의 투자 CSV를 자동 인식하고 표준 구조로 변환 |
| AI 인사이트 자동 생성 | 포트폴리오 분석 후 핵심 인사이트를 텍스트/카드로 자동 제공 |
| 인터랙티브 시각화 | 섹터별, 종목별, 기간별 드릴다운이 가능한 대화형 차트 |
| 자동 리포트 생성 | 분석 결과를 PDF 형태의 투자 리포트로 자동 생성 |

## 핵심 차별화 전략

### "Skills.md = 분석 엔진"

기존 투자 대시보드는 분석 로직이 코드에 하드코딩되어 있어, 새로운 데이터 구조나 분석 기준에 대응하려면 코드를 수정해야 합니다.

AlphaFolio는 분석 규칙을 Skills.md 문서로 분리합니다:
- **분석 규칙 Skills**: 어떤 지표를 계산할지 (수익률, 샤프비율, MDD 등)
- **시각화 규칙 Skills**: 어떤 차트를 어떤 조건에서 보여줄지
- **인사이트 규칙 Skills**: 어떤 패턴이 발견되면 어떤 메시지를 생성할지

이 구조의 장점:
1. **범용성**: Skills.md만 교체하면 주식뿐 아니라 ETF, 채권, 암호화폐 등 다른 자산군에도 대응 가능
2. **바이브코딩 친화**: Skills.md를 AI에게 넘기면 코드가 자동 생성되는 구조
3. **확장성**: 사용자가 직접 분석 규칙을 추가/수정할 수 있는 열린 구조

## 기술 스택

| 영역 | 기술 | 선택 이유 |
|------|------|-----------|
| 프론트/대시보드 | Streamlit | 빠른 프로토타이핑, Python 단일 스택, 바이브코딩 최적화 |
| 데이터 분석 | pandas, numpy | 투자 데이터 가공 및 지표 계산 표준 도구 |
| 시각화 | Plotly, Altair | 인터랙티브 차트, Streamlit 네이티브 지원 |
| AI 인사이트 | LLM API (선택) | 분석 결과 기반 자연어 인사이트 생성 |
| 배포 | Streamlit Cloud | 무료 배포, URL 즉시 공유 가능 |

---

## 데이터 도메인

- **주력**: 국내 주식 + 미국 주식/ETF 포트폴리오
- **지원**: 거래 내역, 보유 종목, 수익률, 섹터/산업 분류
- **데이터 소스**: 더미 데이터 + 공개 데이터 (야후 파이낸스 등)
- **확장 가능**: Skills.md 규칙 교체로 채권, 원자재, 암호화폐 대응 설계
